What is SpanForge?

SpanForge is an adverse event annotation workflow: lexicon + fuzzy weak labeling, structured human curation in Label Studio, provenance-rich gold conversion, and BioBERT foundation for supervised NER.

Features

  • Weak Labeling: Fuzzy + Jaccard gating with negation awareness.
  • Annotation Toolkit: Import, convert, quality metrics, adjudication.
  • Provenance Registry: Trace batches & revisions.
  • Gold Conversion: Canonical mapping, integrity checks.
  • Educational Notebook: End-to-end interactive walkthrough.

Quick Start

conda env create -f environment.yml
conda activate NER
pip install -r requirements.txt
python scripts/verify_env.py
pytest -q

Workflow

1. Ingest

Load complaint text samples (de-identified).

2. Weak Label

Generate candidate SYMPTOM / PRODUCT spans with confidence & negation flags.

3. Annotate

Refine spans in Label Studio for high-quality gold.

4. Convert

Canonical mapping + provenance injection.

5. Quality

Metrics: conflicts, distribution, span density.

6. Train (Planned)

Token classification fine-tune & baseline comparison.
